# 프로젝트 제목

아두이노 보드를 활용한 수발광 센서 라인트레이서 제작

## 요구사항

수광,발광 센서 5세트를 사용하여 정해진 라인을 따라 이동하는 라인트레이서 제작

### 유의점

```
전방에 장애물이 있을 시 정지 상태에서 장애물을 제거하고 다시 경로를 따라 이동하도록 설계
```

### 사용 부품

```
DC모터
서보모터
바퀴
발광센서
수광센서
아크릴 판
아두이노
저항
전선
```
